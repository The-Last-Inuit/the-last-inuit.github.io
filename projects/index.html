A short list of things I build and learn from.


==========================================


DynOSTiC


==========================================


Dynostic is a reusable deterministic tactical-sim core in Rust with a LÖVE (LuaJIT)
front-end. The design goal is fast iteration on UI/feel in LÖVE while keeping the
simulation pure, deterministic, and event-sourced in Rust.
[ rust | lua ]


==========================================


kailinnel


==========================================


a Rust + libp2p project for spinning up ad-hoc “flash institutions”, the kind of
temporary, ownerless coordination layer you want at a hackathon, festival, or pop-up
clinic. No accounts, no servers; the network appears when you need it, disappears
when you don’t.
[ rust ]
